Color Scheme

This is a color scheme that will be used throughout our website in order to enhance our visitors' experience.

Color Usage

#11376A (Prussian Blue)
This is our main color. It will be used as the color for the header and footer. It will also be scattered throughout the website as needed.

#86A5AE (Pewter Blue)
This is our secondary color. It will also be used in the header as the visitor hovers with their mouse over each menu option. It will also be scattered throughout the website as needed.

#FFA400 (Orange Web)
This is our accent color. It will be used as the font color for the menu, and will also be used to highlight certain aspects of the website we believe the visitor should pay attention to.

#F3F6F7 (Cultured)
This color will be used as the website's background color. We will also use it for the font color of our footer.

#080B0C (Rich Black FOGRA 39)
This color will be used as our main font color for headers and paragraphs.
